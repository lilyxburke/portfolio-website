Scroll game

For my first project I decided I wanted to recreate a simple game and landed on the Chrome Dino game as it was both simple and a game I enjoy playing. Using tutorials on YouTube from Game Maker's Toolkit I learnt how to use prefabs, layers and the various components available in the inspector to my advantage.
